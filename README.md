Программа для работы с выпуклыми многоугольниками на плоскости: создание, анализ, проверка свойств, операции над многоугольниками и их визуализация.

## Возможности

- Создание выпуклых многоугольников по координатам вершин
- Вычисление периметра и площади многоугольников
- Проверка принадлежности точки многоугольнику
- Проверка вложенности многоугольников
- Нахождение пересечения многоугольников
- Триангуляция выпуклых многоугольников
- Визуализация многоугольников с помощью matplotlib
- Автоматическая и ручная визуализация операций

### Требования
- Python 3.6 или выше
- pip (менеджер пакетов Python)

### Использование виртуального окружения (venv) с requirements.txt

1. Клонируйте репозиторий:
```bash
git clone <URL репозитория>
cd convex-polygons
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
   - **Windows**:
   ```bash
   venv\Scripts\activate
   ```
   - **macOS/Linux**:
   ```bash
   source venv/bin/activate
   ```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

### Использование uv (альтернатива pip)

Если у вас установлен [uv](https://github.com/astral-sh/uv):

```bash
# Клонирование репозитория
git clone <URL репозитория>
cd convex-polygons

# Создание виртуального окружения и установка зависимостей
uv venv
uv sync

# Активация виртуального окружения
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
```

### Зависимости

Проект использует следующие библиотеки:

```txt
matplotlib>=3.5.0
```

Для корректной работы визуализации также рекомендуется установить:

```bash
# Для лучшей поддержки графиков в Windows
pip install pyqt5

# Или для macOS/Linux
pip install PyQt5
```

### Запуск программы

```bash
python convex_polygons.py
```
